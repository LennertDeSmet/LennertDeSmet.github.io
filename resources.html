LaTeX Templates

I was inspired by Jean-luc Doumont to create a collection of LaTeX templates for presentations, posters and documents that follow his principles for concise and effective communication. This collection is made public on my GitHub for others to enjoy and contribute to under the name Aquila (Latin for eagle).

Making clean and modern presentations in LaTeX beamer can be done with the aquila-beamer template. LaTeX documents with modern typography and layout can be created with the aquila-doc template.

The aquila-poster template is currently being made more usable and will be added later.
